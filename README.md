# Iris Flower Classification

## Objective
Build a Machine Learning model to classify Iris flowers into different species based on sepal and petal measurements.

## Dataset
Iris Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm
- Decision Tree Classifier

## Results
- Accuracy: 100%
- Successfully classified:
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

## Generated Visualizations
- Species Distribution
- Pair Plot
- Confusion Matrix
